# Coding Factory

A local-LLM coding factory CLI for implementing GitHub issues in isolated Docker workspaces.

## Agent Skills

The agent associated with this repository possesses the following skills:

*   Code Generation and Implementation
*   Requirement Analysis and Documentation
*   Git Workflow Management (Branching, Committing, Publishing)
*   Docker Container Orchestration
*   GitHub API Interaction (Issue Fetching, PR Creation)
